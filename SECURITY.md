# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in clanchor, please report it responsibly.

**Email:** valonloshaj@gmail.com

Do not open public issues for security bugs. I'll acknowledge your report within 48 hours and work with you on a fix.

## Scope

clanchor handles GitHub authentication tokens (via `gh` CLI) and writes files to disk. Both are areas where vulnerabilities could have real impact.
